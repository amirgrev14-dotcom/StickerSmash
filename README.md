# Welcome to StickerSmash app 👋

This is a simple, cross-platform image editor that uses stickers that we write in code and stickers that are taken from the rn-emoji-keyboard library.

#Properties
- The app adds stickers to the image.
- Saves the image to your phone or downloads the upload image.
- Stickers can be moved and doubled in size by clicking on the sticker.

That's all there is to it, a simple editor with stickers, you can move them, double their size, and save them either on your computer or on your phone.

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

In the output, you'll find options to open the app in a terminal
